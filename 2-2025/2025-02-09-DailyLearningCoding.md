### 4. Create a Bootstrap Button  
To create a Bootstrap button, simply use the `btn` and `btn-default` classes.  

```html
<button class="btn btn-default">Like</button>
```

---

### 5. Create a Block Element Bootstrap Button  
#### Default Bootstrap Button Behavior  
- When using the `btn` and `btn-default` classes, the button width adjusts to the content inside.  

```html
<button class="btn btn-default">Submit</button>
```

#### Create a Block Element Button with `btn-block`  
- By adding `btn-block`, the button expands to the full width of its container.  
- Elements following it will move to the next line because the button becomes a block element.  

```html
<button class="btn btn-default btn-block">Submit</button>
```

ðŸ‘‰ This button will take up 100% of the containerâ€™s width.  

#### Important Notes When Using `btn-block`  
- The button still requires the `btn` class; otherwise, it will lose Bootstrap styling.  
- It can be combined with other classes like `btn-primary`, `btn-success` to change colors.  
- Useful in mobile-first design, making buttons more visible and easier to click on smaller screens.  

#### Example with Multiple Button Styles  

```html
<button class="btn btn-primary btn-block">Primary Button</button>
<button class="btn btn-success btn-block">Success Button</button>
<button class="btn btn-danger btn-block">Danger Button</button>
```

- `btn-primary` makes the button blue.  
- `btn-success` makes the button green.  
- `btn-danger` makes the button red.
