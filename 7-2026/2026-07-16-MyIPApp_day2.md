# Day 2 – Building the UI with React Components

Today, I started building the **My IP** application using React. Instead of connecting to the API, I used **fake data** to focus on creating the user interface.

I created four components:

* **App** – the root component that stores the fake data.
* **Header** – displays the application title.
* **LocationCard** – displays the IP information.
* **InfoRow** – a reusable component that displays a label and its corresponding value.

I also learned how to pass data from a parent component to a child component using **props**. This helped me understand how React components communicate with each other while keeping the code organized and reusable.

Finally, I added basic CSS to match my UI sketch and completed the first version of the application's layout.
