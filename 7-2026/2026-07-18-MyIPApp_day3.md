# Day 3 – Connecting to a Real API

Today, I updated the **My IP** application by replacing the fake data with data from a real API.

I learned how to use **useState** to store the API response and **useEffect** to fetch data when the application loads. I also used **fetch()** to request data from the API and update the state with the returned response.

Since the component structure was already built in Day 2, I didn't need to change the UI. The existing components automatically displayed the real IP address, country, and country code after receiving the data.

This helped me understand how React retrieves data from an API and automatically updates the user interface when the state changes.
