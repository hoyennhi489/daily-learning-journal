Introduction

This document provides basic knowledge about HTML (HyperText Markup Language) and CSS (Cascading Style Sheets).

HTML is used to structure content on the web.

CSS is used to style and layout web pages.

Understanding both technologies is essential for web development.

Basic Structure of HTML

Here is an example of a basic HTML structure:


            <!DOCTYPE html>
            <html lang="vi">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Technical Documentation</title>
            </head>
            <body>
                <h1>Welcome to Technical Documentation</h1>
                <p>This is an example of a basic HTML structure.</p>
            </body>
            </html>

This example showcases the fundamental elements of an HTML page.

Every HTML document begins with a doctype declaration.

Common HTML Tags

Common HTML tags include:

  • <h1> to <h6>: Heading tags
  • <p>: Paragraph tag
  • <a>: Anchor (link) tag
  • <img>: Image tag
  • <ul>, <ol>, <li>: Unordered and ordered lists

These tags help in structuring content effectively.

Familiarity with these tags is crucial for any web developer.

Introduction to CSS

CSS is used to style and layout HTML elements.

It helps create visually appealing and user-friendly web pages.

CSS can control the layout of multiple web pages all at once.

Basic CSS Properties

Here are some basic CSS properties:


            body {

              background-color: #f0f0f0;

              font-family: Arial, sans-serif;

            }

            

            h1 {

              color: #333;

            }

            

            p {

              font-size: 16px;

            }

These properties help in styling the text and background of web pages.

Linking CSS and HTML

To link a CSS file with an HTML file, use the <link> tag in the <head> section of the HTML document:


            <head>
                <link rel="stylesheet" href="styles.css">
            </head>

This method ensures that the CSS styles are applied to the HTML content.

CSS Box Model

The CSS Box Model represents how HTML elements are structured on a page.

Each element is considered a box that consists of:

  • Content: The content of the element
  • Padding: The space between the content and the border
  • Border: The line surrounding the element
  • Margin: The space between the element and other elements

Understanding the box model is crucial for layout design.

Classification of HTML and CSS Elements

Elements can be classified as:

  • Block elements: Occupy the full width of the page (e.g., <div>, <h1>, <p>)
  • Inline elements: Only occupy space based on their content (e.g., <span>, <a>)
  • Inline-block elements: Combine features of both block and inline elements (e.g., <img>).

This classification helps understand how elements behave in different contexts.

Browser Compatibility

To ensure that your web page works well across different browsers, use standardized CSS properties.

Testing in browsers such as Chrome, Firefox, Safari, and Edge is essential.

Tools like Autoprefixer can improve cross-browser support.
